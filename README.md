# Jason Milne's Personal Website

## Overview

This repository contains the source code for my personal website. The site serves as a professional portfolio, showcasing my skills, projects, and experience in quantitative finance and software development.

## Website Sections

- **Home**: A brief introduction and overview.
- **About**: Detailed information about my background, skills, and interests.
- **Projects**: Showcase of my projects, including:
  - High-Frequency Trading Strategy Development
  - Factor-based Investment Model
  - Options Pricing and Volatility Surface Modeling
- **Resume**: A downloadable version of my professional resume.
- **Contact**: Information on how to reach me.

## Local Development

To run this website locally:

1. Clone the repository: `git clone https://github.com/jasonamilne/jasonamilne.github.io.git`
2. Navigate to the project directory
3. Open `index.html` in your web browser

## Deployment

This website is deployed using GitHub Pages and can be accessed at: https://jasonamilne.github.io
